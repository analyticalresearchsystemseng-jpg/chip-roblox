Installation Steps

  1. Click the blue button above to download
  2. Open Roblox Studio
  3. Go to Plugins → Install Plugin From File...
  4. Select the downloaded ChipBridge.rbxmx
  5. Done! Look for the "Chip Bridge" toolbar button

How It Works

📝 Ask Chip in Telegram
💻 Chip writes Lua code
🎮 Click "Get from Chip" in Studio
✨ Code appears in your project!
